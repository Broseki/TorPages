About

This service is freely provided by CaveFox Telecommunications; we believe
that everyone has a right to be heard without fearing for their safety.

This system allows people to publish static content to the Deep Web without
having to worry about the security, and technical parts of the site.

When you publish a page you simply paste your code into the appropriately
labeled box on the home page. If you want a custom link
(eg. http://m54wkp5ctdpummms.onion/p/custom_link_here) enter it in the fourth
box labeled "Custom Post ID". IDs must only contain letters and numbers, and
will be converted to lower case. If the post is accepted you will be given a
modification key, SAVE THIS! The modification key can be entered into the box
labeled "Post Modification Key" to modify the page.

To modify a post simply enter the updated code to the appropriately labeled box,
enter your post ID into the box labeled "Post ID", enter your modification key into
the box labeled "Post Modification Key", and leave the box labeled "Custom Post ID"
blank, click submit and your page will be updated. A NEW MODIFICATION KEY WILL BE
GENERATED, the old key will be made invalid in order to better secure your site.

Follow the rules and your site will be kept up for as long
as the service is offered. We follow US law in order to keep the service available
for those that want to be heard, but are restricted from doing so in some way
whether it be financial difficulties, or other complications.

Please note that while we obey US laws we are not technically capable of logging IP
addresses that connect to the server due to the use of Tor. If you see rule violations
email cavefox@tuta.io and the report will be reviewed. PLEASE DO NOT ASK FOR IP LOGS,
WE WOULD NOT EVEN BE TECHNICALLY CAPABLE OF KEEPING THEM IF WE WANTED TO.
