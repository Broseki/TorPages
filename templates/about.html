About

This service is freely provided by CaveFox Telecommunications; we believe
that everyone has a right to be heard without fearing for their safety.

This system allows people to publish static content to the Deep Web without
having to worry about the security, and technical parts of the site.

Create an account by clicking register on the home page; if you have
an accout just login. You will be directed to the user management page,
the rest is pretty straight forward.

When you publish a page you simply paste your code into the appropriately
labeled box on the create post page. If you want a custom link
(eg. {{ site_url }}/p/custom_link_here) enter it in the box labeled
"Custom Post ID". IDs must only contain letters and numbers.

To modify a post simply select edit in the management console next to the page's
name, and modify the code to what you want; then just click publish.

Follow the rules and your site will be kept up for as long
as the service is offered. We follow US law in order to keep the service available
for those that want to be heard, but are restricted from doing so in some way
whether it be financial difficulties, or other complications.

Please note that while we obey US laws we are not technically capable of logging IP
addresses that connect to the server due to the use of Tor. If you see rule violations
email {{ admin_email }} and the report will be reviewed. PLEASE DO NOT ASK FOR IP LOGS,
WE WOULD NOT EVEN BE TECHNICALLY CAPABLE OF KEEPING THEM IF WE WANTED TO.
